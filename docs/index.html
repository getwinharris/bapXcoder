Frequently Asked Questions

Do I need internet after downloading?
Yes, initial authentication and download require internet. After download, the IDE runs completely offline with all AI functionality.
Where is my project data stored?
Each project gets its own .bapXcoder directory with session.json and todo.json files. All data stored locally on your machine.
How does this differ from cloud IDEs?
This is a local IDE that runs on your machine with project-based memory. No session loss, no quotas, and full privacy after download.
