# bapXcoder - Professional Landing Page

This repository includes a professional landing page for bapXcoder, an AI-powered IDE with project memory features.

## Features
- Project-based memory system 
- AI-driven testing and validation
- VS Code style interface
- Desktop PWA installation
- Free and open source

## GitHub Pages
This site is hosted at: https://getwinharris.github.io/bapXcoder/

## Design Philosophy
- Clean, professional appearance
- Emphasis on project memory features
- Comparison with traditional paid tools
- Clear download CTA

The landing page is designed to attract developers looking for an alternative to expensive AI tools with persistent project context.
